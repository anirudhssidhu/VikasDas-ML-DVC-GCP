1. Create a repo on Github with readme.md, etc
2. Clone the repo in the local
3. create a vitural env .venv
    - py -3.11 -m venv .venv
    - source .venv/Scripts/activate
    - python -m pip install --upgrade pip

4. create a requirements.txt file for libraries
5. pip install -r requirements.txt
6. Create two folders src and experiments
7. Experiments folder - 
    - notebook.ipynb
    - spam.csv

8. In the scr folder/ data_ingestion.py:
    - write the code for ingestion
    - python src/data_ingestion.py

9. In the src folder/ data_preprocessing.py:
    - write the code for data_preprocessing
    - python src/Pre-processing.py

10. In the src folder/ feature_engineering.py:
    - write the code for feature_engineering
    - python src/feature_engineering

11. In the src folder/ model_building.py
    - write the code for model_building
    - python src/model_building.py

12. In the src folder/ model_evaluation.py
    - write the code for model_evaluation
    - python src/model_evaluation.py

13. Add data, models, reports, logs directories to .gitignore file:
    - git add . / git commit -m "pipeline, models, reports, logs completed" / git push origin main

14. Setting up dvc pipeline (without params):
    - create dvc.yaml file and add stages to it
    - dvc init 
    - dvc repro (test pipeline automation)
    - dvc dag
    - git add/commit/push

15. Setting up the dvc pipeline with params:
    - add params.yaml file
    - add the params setup (in each file)
    - add params to dvc.yaml file 
    - run "dvc repro" to test the pipeline with params
    - git add/commit/push
